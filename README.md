# Reza V11 Champagne Luxury

Fresh rebuild. Soft luxury, champagne, ivory, rose-gold style.

## Run locally

```bash
cd ~/Downloads
unzip -o reza-v11-champagne-luxury.zip
cd reza-v11-champagne-luxury
chmod +x setup.sh
./setup.sh
```

Then use 3 terminals:

```bash
cd backend && npm start
```

```bash
cd frontend && python3 -m http.server 5173
```

```bash
cd admin && python3 -m http.server 5174
```

Admin login: admin@reza.co.za / reza2026

## Render
Backend Web Service root: backend. Frontend Static Site root: frontend. Admin Static Site root: admin.
